## Site content

This project reads your profile and links from `src/consts/homePageData.ts`. That file is ignored by Git so your personal details stay local.

To configure the site:

1. Copy or rename `src/consts/homePageData-sample.ts` to `src/consts/homePageData.ts`.
2. Edit `homePageData.ts` and replace the sample values with your real name, links, experience, projects, and anything else the exports define.

Until `homePageData.ts` exists, the app will not resolve those imports—create it from the sample file before running or building.
